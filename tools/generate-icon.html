Merge Monsters Icon Generator

端末にインストール後のアイコンを差し替えるには、まず下の3枚をダウンロードし、 プロジェクトの resources/ フォルダーに以下の名前で保存してください。

  • icon.png ... 1024×1024 の通常アイコン (旧端末・ストア用)
  • icon-foreground.png ... Adaptive Icon の前景 (Android 8.0以降)
  • icon-background.png ... Adaptive Icon の背景

その後ターミナルで npm run icons:generate → npx cap sync android を実行すると、 必要な全サイズが自動生成され、Androidプロジェクトに反映されます。
